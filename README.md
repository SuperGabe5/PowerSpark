# PowerSpark

This is a GTK program that allows for me to make PowerForge accually be able to be installed because i have no maven hosting

it supports Windows (x86 and x64), macOS (M1 or Better) and Linux (amd64 and arm64)

> [!NOTE] 
> System Requirements:
>
> CPU: Anything that is at least middle class that was Made in 2012 or later
>
> RAM: 6 GB of FREE RAM (Good Luck users of Windows 10 22H2 or later)
>
> Disk Space: AT LEAST 6.0 GB FREE
